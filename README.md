# Extensions

-   Graphical User Interface for the client

## Comment tester les projets

1. Lancer le projet "WcfService"
2. Lancer le projet "WcfGui" pour tester le service avec une interface graphique ou bien lancer "Console-Velib" pour tester le service en mode console

## Comment utiliser WcfGui

L'interface graphique permet de saisir deux champs. Le premier correspond à la station et le second à la ville. Après avoir saisis de la bonne manière les champs, il faut appuyer sur le bouton correspond à droite et le résultat s'affichera.

## Comment utiliser Console-Gui

La console propose un certain nombre de commandes. Les voicies :
-  **Exit :** : ferme le programme
-  **Help :** affiche  la description des différentes commandes
- **Station *ville* :** affiche les stations de la *ville* saisie
- **Available_Bikes *vile* *station* :** affiche le nombre de vélos disponibles pour la ville et station indiquées
